Privacy Policy

Effective Date:28/10/2025

1. Data Collection

The Tic Tac Toe extension does not collect, store, or share any personal information from users. The game operates entirely within your browser and does not send data to external servers.

2. Permissions

The extension only uses the permissions required to display and run the game locally. It does not access or track browsing history, personal information, or user activity.

3. Third-Party Services

This extension does not use third-party services, analytics tools, or advertising networks.

4. Data Security

Because no personal data is collected, there is no risk of unauthorized access or data breaches related to user information.

5. Children's Privacy

This extension is safe for all ages and does not collect information from children or adults.
